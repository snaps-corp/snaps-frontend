## Snaps FE 과제 안내
```
├── README.md
├── guide
│   ├── Guide.pdf
│   ├── animation.mov
│   ├── default-screen.png
│   ├── options.json
│   └── showcase.json
├── package.json
├── public
└── src
```
- guide
    - animation.mp4: 과제 예시입니다.
    - options.json:  '옵션 영역' 구현시 필요한 옵션 정보 입니다.
    - showcase.json: '쇼케이스 영역' 구현시 필요한 정보 입니다.
- src: (컴포넌트 구성 등 자유롭게 작성해 주세요.)
    - README.md: (과제와 관련하여 특별하게 이야기 하고 싶은 부분을 적어주세요.)
